# Student Performance Prediction using Linear Regression 

This project shows a pipeline of machine learning regression using Python and Scikit-learn. The goal of this project is to predict the student marks on the basis of study hours and sleep duration.
 ## Project Overview
 In this project, we:
 - Created a dataset using Pandas
 - Applied train-test split
 - trained using Linear Regression
 - Generated Predictions
 - Evaluated performance of the model using MSE and R^2 score
 - Analyzed model accuracy and reliability using residual plots
 - Interprected the model coefficients of its line
This represents the basic usage of Linear Regression pipeline.
---

## Dataset Description

The dataset includes features of students:

- Hours Studied 
- Sleep Hours
- Marks Scored

- These features helps us to understand the pattern or habits that affects the academic performance.
